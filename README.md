# Projeto

## Exemplo Java Básico

Este repositório contém um exemplo simples em Java desenvolvido para fins de aprendizado.

## Descrição
O programa exibe uma mensagem de "Olá, mundo!" no console.

## Tecnologias
- Java

## Como executar
1. Compile o arquivo:
   javac OlaMundo.java
2. Execute:
   java OlaMundo

   public class OlaMundo {
    public static void main(String[] args) {
        System.out.println("Olá, mundo!");
    }
}
